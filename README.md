# Spring_security_app

This small, simple project is designed to practice and use skills.
The program supports all CRUD operations. Employee data is received and written to a local database.


Technology stack:
- Spring MVC
- Spring Security
- JSTL
- Hibernate 
- MySQL
- Lombok
- Tomcat
<br>
- Service Layer, DAO
<br>
- XML-file configuration
